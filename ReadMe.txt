Странички для создания фонового изображения программы iiko-front на экране блокировки с отображением CRM ID и преобразованием этого ID в QR-код. Пользователь сканирует QR-код и переходит в WhatsApp технической поддержки с предварительно заполненным CRM ID, после чего может отправить сообщение. Это позволяет быстро определить, из какого заведения обращается клиент.

* Файл index.html позволяет ввести CRM ID в соответствующее поле и загрузить готовое изображение.
* Файл crmlink.html предоставляет прямой доступ к странице через ссылку с переданным в адресной строке CRM ID. Пример ссылки: http://localhost/iiko/crmlink.html?text=1234567, где последние цифры — это CRM ID. Этот подход может быть удобен при создании ссылок из Excel.

В целях правильного отображение во многих POS моноблоках, размер изображения выбран 1366х768px. Вы можете создать свое изображение и заменить существующее в папке img, название должно совпадать. 

Для изменения номера WhatsApp откройте указанные файлы блокнотом или любым редактором и внесите соответствующие правки, приведен комментарий в строке 115 (index) и 119 (crmlink).

!!!Странички не работают автономно как файлы, их нужно целиком развернуть в любом сервере и обращаться через браузер. В моем случае, запущен Denwer и адрес localhost.
